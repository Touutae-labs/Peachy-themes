# Peachy Themes for VS Code

Peachy Themes is the main VS Code package in the Peachy Collection.

It keeps the original cute peach-and-rose mood, but now uses a more consistent syntax system with language-aware colors for:

- TypeScript and JavaScript
- TSX and JSX
- JSON and YAML
- HTML and CSS
- Markdown
- Python
- Shell scripts
- Rust
- Go

## Install

Download it from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Piros-k.peachy-themes&ssr=false#overview).

## What is themed

- editor surface, tabs, panels, and status bar
- syntax tokens and semantic tokens
- diagnostics, diff colors, and terminal ANSI colors
- language-oriented defaults for TS, JS, React, JSON, YAML, Markdown, Python, shell, Rust, and Go

## Recommended settings

Ready-made settings are in [examples/peachy-language-settings.json](./examples/peachy-language-settings.json).

Language notes and supported scopes are in [docs/language-support.md](./docs/language-support.md).

## Packaging

This package is what gets bundled into the Marketplace `.vsix` build from the release workflow.

## Origins

Peachy Themes was inspired by [PinkCatBoo](https://github.com/ftsamoyed/PinkCatBoo), then adapted into its own softer Peachy palette.
