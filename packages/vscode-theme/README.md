# Peachy Themes for VS Code

A sweet, cozy dark theme family with two distinct flavors. Pick the one that matches your mood.

## Theme Variants

### Peachy Themes - Pink (Legacy)

The original vibrant theme that started it all. Hot pinks, golden yellows, lavender types, and bold green keywords give your editor an energetic candy aesthetic.

- Keywords: `#FF4791` (hot pink) / `#CDEAC0` (mint)
- Strings: `#FAE8B6` (warm cream) / `#FF928B` (peach)
- Functions: `#FFC85B` (golden)
- Types: `#F5A6C6` (pink) / `#DCBFF2` (lavender)
- Comments: `#729760` (sage green)

### Peachy Themes - Peach (New)

A softer, more cohesive variant built from the Peachy design token system. Every color traces back to a single palette definition, giving you a calmer, more unified experience.

- Keywords: `#cdeac0` (mint)
- Strings: `#ff928b` (peach)
- Functions: `#efe9ae` (cream)
- Types: `#fec3d6` (soft pink)
- Comments: `#729760` (sage green)
- 210 UI colors with full semantic highlighting

## Install

Download from the [Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=Piros-k.peachy-themes).

Or install from the command line:

```
code --install-extension Piros-k.peachy-themes
```

## Switching Themes

After installing, open the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`), search **Color Theme**, and choose either:

- **Peachy Themes - Pink** — the original
- **Peachy Themes - Peach** — the new softer variant

## What's Themed

Both variants cover:

- Editor surface, tabs, panels, status bar, and terminal
- Syntax tokens for JavaScript, TypeScript, JSON, YAML, Markdown, Python, Shell, Rust, Go, and more
- Diagnostics, diff colors, and terminal ANSI colors

The Peach variant additionally includes:

- Semantic highlighting for classes, decorators, enums, interfaces, and more
- Bracket pair colorization using the Peachy palette
- Extended inlay hint, notification, and widget styling
