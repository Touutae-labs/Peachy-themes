# Peachy WezTerm

This package contains the Peachy terminal color palette for WezTerm.

## Usage

Import `peachy.lua` into your WezTerm configuration and register it as a color scheme.

The palette keeps the same Peachy background, peach accent, cream highlight, and soft mint/comment contrast used across the rest of the collection.
