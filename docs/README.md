# Peachy Docs

This folder is the documentation hub for the Peachy Collection.

Start here if you want the shortest path to the project shape:

- [Repo structure](./repo-structure.md)
- [Publishing guide](./publishing.md)
- [Brand guide](./brand.md)

## What belongs here

- contributor-facing setup and release notes
- package usage notes that are broader than one README
- brand and asset guidance
- repo-wide structure and publishing flow

## What stays in package READMEs

- install commands
- short package-specific usage notes
- package-level links to deeper docs
