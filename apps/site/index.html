Two flavors, one family

A cozy dark theme for your whole dev setup.

Peachy ships two variants — Pink (vibrant and bold) and Peach (soft and calm) — across VS Code, Neovim, WezTerm, and Oh My Posh. Pick one mood and use it everywhere.

Install for VS Code Read the Docs 
// Two variants in every port
const theme = {
  pink:  "vibrant, bold, energetic",
  peach: "soft, calm, cohesive",
};
// VS Code · Neovim · WezTerm · Oh My Posh
