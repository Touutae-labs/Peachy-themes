Theme family

Soft, sweet tooling for people who want their setup to feel alive.

Peachy starts with the original VS Code theme and grows into a full collection: editor ports, shell prompts, documentation, and a shared palette you can keep tuning.

Install VS Code Theme Open GitHub Open Playground 
const peachy = {
  background: "#202330",
  peach: "#ff928b",
  cream: "#efe9ae",
  mint: "#cdeac0"
}

function sweeten(editor) {
  return editor.apply(peachy)
}

Collection

One palette, multiple homes.

VS Code

The original Peachy extension, upgraded as the flagship package in the collection.

LazyVim

A starter Neovim colorscheme that keeps the same dark base, sweet pinks, and cream highlights.

Oh My Posh

A matching shell prompt so your terminal feels like part of the same visual world.

Tokens + Generator

Shared tokens and generated assets make future ports easier and keep color decisions consistent.

Palette

The Peachy core.

Install

Grow the collection from one repo.

Start with the VS Code theme, then expand the same mood into your editor, prompt, and docs site. The collection is organized so each port can become its own package or repo later. 

git clone git@github.com:Touutae-labs/Peachy-themes.git
cd Peachy-themes
node scripts/color-audit.mjs
